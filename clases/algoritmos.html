Unidad 2

Utilización de estructuras repetitivas "Ciclos" y subprogramas

Introducción

Las estructuras repetitivas, también conocidas como ciclos o bucles, son fundamentales en la programación ya que permiten ejecutar un conjunto de instrucciones múltiples veces sin necesidad de escribir el código repetidamente. Junto con los subprogramas, forman la base para crear programas eficientes y modulares.

Tipos de Ciclos

1. Ciclo For (Para)

Se utiliza cuando conocemos el número exacto de iteraciones que queremos realizar. Es ideal para recorrer arreglos o ejecutar un bloque de código un número determinado de veces. 

Para i desde 1 hasta 10 hacer
    Escribir "Iteración número: ", i
Fin Para
2. Ciclo While (Mientras)

Se ejecuta mientras una condición sea verdadera. Es útil cuando no sabemos exactamente cuántas veces necesitamos repetir el código. 

contador = 0
Mientras contador < 5 hacer
    Escribir "Contador: ", contador
    contador = contador + 1
Fin Mientras
3. Ciclo Do-While (Hacer-Mientras)

Similar al While, pero garantiza que el código se ejecute al menos una vez, ya que la condición se evalúa al final. 

Hacer
    Escribir "Ingrese un número positivo: "
    Leer numero
Mientras numero <= 0

Subprogramas

Los subprogramas son bloques de código reutilizables que realizan tareas específicas. Se dividen principalmente en dos tipos:

Funciones

Son subprogramas que devuelven un valor después de realizar su tarea. Se utilizan cuando necesitamos calcular y retornar un resultado. 

Función calcularPromedio(num1, num2, num3)
    suma = num1 + num2 + num3
    promedio = suma / 3
    Retornar promedio
Fin Función
Procedimientos

Son subprogramas que ejecutan una serie de instrucciones pero no devuelven un valor. Se utilizan para realizar tareas específicas sin necesidad de retornar resultados. 

Procedimiento mostrarMenu()
    Escribir "1. Opción 1"
    Escribir "2. Opción 2"
    Escribir "3. Salir"
Fin Procedimiento

Funciones en C++

En C++, las funciones son bloques de código reutilizables que realizan tareas específicas. Una función puede recibir datos (parámetros), procesarlos y devolver un resultado. C++ ofrece diferentes formas de pasar parámetros a las funciones.

Sintaxis Básica de una Función
tipo_retorno nombre_funcion(tipo_parametro1 parametro1, tipo_parametro2 parametro2) {
    // Cuerpo de la función
    return valor; // opcional
}
Tipos de Parámetros en C++
1. Paso por Valor

Se crea una copia del valor original. Los cambios en la función no afectan la variable original.

#include <iostream>
using namespace std;

void incrementarPorValor(int x) {
    x = x + 1;
    cout << "Dentro de la función: " << x << endl;
}

int main() {
    int numero = 5;
    cout << "Antes: " << numero << endl;
    incrementarPorValor(numero);
    cout << "Después: " << numero << endl; // Sigue siendo 5
    return 0;
}
2. Paso por Referencia (&)

Se pasa la dirección de memoria de la variable. Los cambios en la función SÍ afectan la variable original.

#include <iostream>
using namespace std;

void incrementarPorReferencia(int &x) {
    x = x + 1;
    cout << "Dentro de la función: " << x << endl;
}

int main() {
    int numero = 5;
    cout << "Antes: " << numero << endl;
    incrementarPorReferencia(numero);
    cout << "Después: " << numero << endl; // Ahora es 6
    return 0;
}
3. Paso por Puntero (*)

Se pasa la dirección de memoria usando punteros. Permite modificar la variable original.

#include <iostream>
using namespace std;

void incrementarPorPuntero(int *x) {
    *x = *x + 1;
    cout << "Dentro de la función: " << *x << endl;
}

int main() {
    int numero = 5;
    cout << "Antes: " << numero << endl;
    incrementarPorPuntero(&numero);
    cout << "Después: " << numero << endl; // Ahora es 6
    return 0;
}
Ejemplo Completo: Calculadora con Referencias

Este ejemplo muestra cómo usar referencias para devolver múltiples valores desde una función:

#include <iostream>
using namespace std;

// Función que calcula múltiples operaciones usando referencias
void calculadora(double a, double b, double &suma, double &resta, 
                double &multiplicacion, double &division) {
    suma = a + b;
    resta = a - b;
    multiplicacion = a * b;
    
    if (b != 0) {
        division = a / b;
    } else {
        division = 0; // Evitar división por cero
        cout << "¡Advertencia: División por cero!" << endl;
    }
}

// Función para intercambiar dos valores usando referencias
void intercambiar(int &x, int &y) {
    int temp = x;
    x = y;
    y = temp;
}

int main() {
    // Ejemplo 1: Calculadora
    double num1 = 10.0, num2 = 3.0;
    double resultado_suma, resultado_resta, resultado_mult, resultado_div;
    
    calculadora(num1, num2, resultado_suma, resultado_resta, 
               resultado_mult, resultado_div);
    
    cout << "=== CALCULADORA ===" << endl;
    cout << num1 << " + " << num2 << " = " << resultado_suma << endl;
    cout << num1 << " - " << num2 << " = " << resultado_resta << endl;
    cout << num1 << " * " << num2 << " = " << resultado_mult << endl;
    cout << num1 << " / " << num2 << " = " << resultado_div << endl;
    
    // Ejemplo 2: Intercambio
    int a = 5, b = 10;
    cout << "\n=== INTERCAMBIO ===" << endl;
    cout << "Antes: a = " << a << ", b = " << b << endl;
    intercambiar(a, b);
    cout << "Después: a = " << a << ", b = " << b << endl;
    
    return 0;
}
Ventajas de usar Referencias
  • Eficiencia: No se crean copias de los datos, especialmente útil con objetos grandes.
  • Modificación directa: Permite que la función modifique las variables originales.
  • Múltiples valores de retorno: Una función puede "devolver" varios valores modificando parámetros por referencia.
  • Sintaxis más limpia: Más fácil de leer que los punteros (no necesita * ni &).

Aplicación Práctica

La combinación de ciclos y subprogramas permite crear programas más organizados y eficientes. Por ejemplo, podemos usar un ciclo para procesar múltiples datos y una función para realizar cálculos específicos en cada iteración.

Ejemplo: Calculadora de promedios
Función calcularPromedio(numeros[], cantidad)
    suma = 0
    Para i desde 0 hasta cantidad-1 hacer
        suma = suma + numeros[i]
    Fin Para
    Retornar suma / cantidad
Fin Función

Algoritmo Principal
    Escribir "¿Cuántos números desea ingresar?"
    Leer cantidad
    
    Dimension numeros[cantidad]
    
    Para i desde 0 hasta cantidad-1 hacer
        Escribir "Ingrese el número ", i+1, ":"
        Leer numeros[i]
    Fin Para
    
    promedio = calcularPromedio(numeros, cantidad)
    Escribir "El promedio es: ", promedio
Fin Algoritmo

Conclusión

El dominio de las estructuras repetitivas y los subprogramas es esencial para cualquier programador. Estas herramientas nos permiten escribir código más limpio, eficiente y mantenible, evitando la repetición innecesaria y promoviendo la modularidad en nuestros programas.

Problema Resuelto: Sistema de Análisis de Audiencia Televisiva

Contexto: Una famosa cadena de televisión desea un programa que le permita conocer el tipo de audiencia que tiene en diferentes franjas horarias.

Franjas Horarias:
  • Franja 1: Antes de las 14:00h
  • Franja 2: De 14:00h a 20:00h
  • Franja 3: De 20:00h a 22:00h
  • Franja 4: Después de las 22:00h
Grupos de Edad:
  • Grupo 1: Menos de 30 años
  • Grupo 2: 30 años o más

Funciones Implementadas

🔹 Incluir(A:AUDIENCIA; edad:entera; hora:T_hora)

Actualiza la estructura de audiencia con los datos de un televidente.

🔹 Consulta(A:AUDIENCIA; franja:entera)

Devuelve el número de televidentes de una franja específica.

🔹 Máximo(A:AUDIENCIA)

Devuelve la franja horaria con mayor audiencia.

🔹 Conv_hora(h:T_hora)

Función auxiliar que convierte hora a franja horaria.

🔹 Conv_edad(e:entera)

Función auxiliar que convierte edad a rango etario.
